## Как проверить визуально

1. **Запуск:** из папки `webdev-exam`: `python run.py`, открыть http://127.0.0.1:5000

2. **Проверить по страницам:**
   - **Главная** — `/`: hero, блок преимуществ, каталог, карточки товаров, пагинация (если страниц несколько).
   - **Товар** — клик по товару: фото, цена, описание, кнопка «В корзину», вкладка «Описание».
   - **Корзина** — `/cart`: таблица, изменение количества, удаление, блок «Итого» справа, «Оформить заказ».
   - **Оформление заказа** — после перехода из корзины (нужен авторизованный пользователь): форма слева, summary справа.
   - **Мои заказы** — `/orders` (вход обязателен): список заказов в карточках.
   - **Контакты** — `/contacts`: hero и три карточки.
   - **Корпоративные заказы** — `/corporate`: hero, форма и шаги 1–2–3.
   - **Вход** — `/auth/login`: карточка по центру.
   - **Регистрация** — `/auth/register`: карточка с полями.
   - **Админка** — `/admin` (под учёткой админа): дашборд, категории, товары, заказы, заявки, импорт CSV.

3. **Проверить адаптив:** сузить окно — navbar сворачивается в бургер, карточки и колонки перестраиваются (1 колонка на мобильных).

4. **Проверить скролл:** при прокрутке вниз navbar получает класс `scrolled` и более плотный фон (см. `app.js`).

## Палитра и стиль

- **Primary:** глубокий зелёный `#1b4332` (кнопки, ссылки, акценты).
- **Primary-2 / светлый зелёный:** `#40916c`, `#d8f3dc` (hover, бейджи, фоны).
- **Акцент (мёд/золото):** `#b8860b` (опционально для CTA).
- **Фон:** тёплый off-white `#faf8f5`.
- **Карточки:** белый фон, мягкие тени, скругления 16px.
- **Шрифты:** Manrope (текст), Playfair Display (заголовки).
